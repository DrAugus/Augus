# Augus

## Develop

prerequisites：

- nodejs(Version >= 16)
- yarn(Version >= 1) (`npm install -g yarn`)

git clone：

```git
git clone https://github.com/DrAugus/Augus.git
```

run：

```yarn
yarn install
yarn docs:dev
```
