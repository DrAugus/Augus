# 复杂度

| | 查找 | 插入 | 删除|
|-|-|-|-|
|数组 | O(n)| O(1) | O(n)|
|有序数组 | O(logn)| O(n)| O(n)|
|链表 | O(n)| O(1)  | O(n)|
|有序链表 | O(n)| O(n)| O(n)|
|二叉树最坏 | O(n)| O(n)| O(n)|
|二叉树一般 | O(logn)| O(logn)| O(logn)|
|平衡树 | O(logn)| O(logn) | O(logn)|
|哈希表 | O(1)| O(1)| O(1)|
